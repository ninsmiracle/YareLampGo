# Hardware Guide

This folder contains public hardware documentation for YareLampGo. It is meant
to help community builders understand the module layout, wiring, and assembly
path without exposing supplier production files or private manufacturing
documents.

## Files

| File | Purpose |
| --- | --- |
| [wiring.md](wiring.md) | Public wiring table converted from the uploaded hardware wiring spreadsheet. |
| [hardware-wiring.png](hardware-wiring.png) | Wiring diagram for the current public hardware layout. |
| [component-flatlay.jpg](component-flatlay.jpg) | Component flat lay / public hardware preview. |
| [../hardware-and-assets-scope.md](../hardware-and-assets-scope.md) | License and publication boundary for hardware, printable files, and visual assets. |
| [../../assets/printable/README.md](../../assets/printable/README.md) | Public printable appearance and structural files. |

## Publication Boundary

The files in this folder are public documentation. They do not include supplier
quotations, private part numbers, factory process files, or production CAD.
Printable appearance and structural files live under `assets/printable/` and
have their license declared in `ASSET_LICENSES.md`.
