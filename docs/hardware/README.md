## 重要边界

电路 PNG 用于理解网络、接口、器件位置和 PCB 走线，不是 Gerber/钻孔/坐标/生产 BOM。完整 STEP 是装配参考总成，不是已经拆好的逐件 STL/3MF。任何制造、制板和首次上电都需要制作者自行完成工程复核。

具体授权以仓库根目录的 [ASSET_LICENSES.md](../../ASSET_LICENSES.md) 为准。
