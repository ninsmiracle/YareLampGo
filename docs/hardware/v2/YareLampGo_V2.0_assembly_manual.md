## 四、组装过程

### 1. 电机外壳组装

![电机外壳组装 1](assembly-manual-media/image30.jpeg)

![电机外壳组装 2](assembly-manual-media/image31.jpeg)

![电机外壳组装 3](assembly-manual-media/image32.jpeg)

- 舵机线缆应置于外壳内部，避免被挤压导致外壳无法完全装到位。
- 三处舵机的线缆走向保持一致。
- 写入舵机 ID 或更换线缆前必须断开 12V；写 ID 时总线上只能连接一颗未编号舵机。具体 ID 对应关系见 [V2.0 接线指南](../wiring.md)。

## 首次上电前必查

原 DOCX 是机械组装参考；首次上电前请额外完成以下 V2.0 检查：

1. 不混用 V1/V2 结构件、接线图或校准文件。
2. 确认 12V 舵机侧和 +5V 逻辑侧没有接反，所有逻辑地共地。
3. S3 GPIO43 TX → C6 RX，S3 GPIO44 RX ← C6 TX；LED U3 针序以 PCB 丝印和实测方向为准。
4. 扶稳机构、清空运动范围后，先运行 `uv run lampgo detect`、`uv run lampgo scan-motors --ids 1-5` 和 `uv run lampgo ping`，确认五颗舵机稳定在线后再校准。

完整手动软件、烧录与校准流程见 [V2.0 手动安装、烧录与首次启动](../../getting-started/manual-hardware-setup.md)。
