# 项目说明

`lampgo` 想解决一个很朴素的问题：机械臂和具身智能很好玩，但普通人太难上手了。

过去一台 5 自由度机械臂更像实验室里的设备。它能动，但你得懂电机、串口、校准、安全限位、动作规划和一堆底层细节，才能让它做出一个看起来还不错的动作。对普通软件开发者、自媒体创作者和玩家来说，这个门槛太高。

`lampgo` 把这些底层能力打包成一个本地运行时，让机械臂台灯变成更容易使用的桌面小伙伴：它可以听你说话，看见环境，自己动起来，用灯光、动作和语音回应你。

## 我们最看重的用户价值

- **降低使用门槛**：用户不用先成为机器人专家，就能通过网页、命令行、自然语言或 Agent 控制机械臂。
- **降低创作门槛**：普通开发者可以像调用软件 API 一样调用电机、LED、摄像头和语音能力。
- **降低内容门槛**：创作者可以录制动作、播放表情、设计互动，让桌面硬件更适合短视频、直播和 Demo。
- **降低实验门槛**：硬件团队可以更快验证“有身体的 AI”在桌面场景里到底能做什么。

## 它不是只会聊天的 AI

`lampgo` 的重点不是再做一个聊天框，而是让 AI 能产生真实的身体反馈：

- 用户说“点个头”，台灯真的点头。
- 用户说“做个害羞的表情”，它可以配合动作和 LED 表情回应。
- 用户录制一个动作，下次可以用自然语言、Web UI 或 Agent 再次触发。
- Agent 可以读取摄像头画面，判断桌面环境，再调用动作和表情工具。

## 它把哪些能力接在一起

| 能力 | 对用户意味着什么 |
| --- | --- |
| 运动控制 | 机械臂能安全、平滑地动起来。 |
| 动作录制 | 普通人可以用手摆出动作，保存后反复播放。 |
| LED 表情 | 台灯不只会动，还能用灯光表达状态和情绪。 |
| 摄像头 | 台灯可以看到桌面或用户所在环境。 |
| 语音 | 用户可以说话，台灯也可以语音回应。 |
| 触摸反馈 | 台灯头部触摸电容可以把摸头、轻拍等接触反馈传回 PC，让交互更像真实陪伴。 |
| Web UI | 不写代码也能操作、录制、配置和调试。 |
| OpenClaw 插件 | 复杂玩法可以交给 Agent 规划和执行。 |

## 适合探索的场景

- 桌面陪伴机器人
- 短视频和直播互动道具
- 会动的 AI 助手
- 智能硬件原型 Demo
- Agent 控制真实世界设备
- 面向普通开发者的具身智能创作平台
