日志
业务事件（意图 / 技能 / 工具调用 / TTS 等）。每次刷新都会从 ~/.lampgo/events.log 回放最近 200 条，所以不会丢。
UI 日志 = 面向用户的业务事件，写入 ~/.lampgo/events.log（JSONL，单段 10MB，轮转 3 份）。 终端日志 = 进程 stderr，面向开发者，粒度更细（motor bus、LLM 原始请求等）。 两者不重复，视角不同。
