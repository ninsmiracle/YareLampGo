当前设备在线，需要先解绑并重置 WiFi 才能重新配网。
  1. 点击下方「重置设备并继续」，系统会解除当前电脑配对并清除设备 WiFi，设备将自动重启并进入配网模式。
  2. 如果设备不在电脑旁（无法通过网络操作），也可以长按 ESP32 上的 BOOT 按钮 10 秒手动重置。
首次使用或重置网络后，ESP32 会开启一个临时 WiFi 热点，名字形如 Lampgo-Setup-XXXX（XXXX 为设备 MAC 末 4 位，印在卡片上或串口日志里）， 默认密码 lampgo123。
  1. 用手机 / 笔记本连上该热点。
  2. 系统可能会弹出"无互联网连接"的提示，保持连接即可。
  3. 连上后回到这个页面，点击下一步。一般不需要手动打开其它网页。

找不到 Lampgo-Setup-XXXX？长按 ESP32 BOOT 按钮 10 秒可重置 WiFi 并重新进入配网模式。

选择要让台灯连接的 2.4GHz WiFi（设备暂不支持 5GHz），输入密码后点击“发送”。 配置成功后，台灯会一直连接这个 WiFi；以后电脑上的 lampgo 后端也会通过这个 WiFi 找到并连接台灯。 发送后设备会关闭临时热点，并尝试连接你选择的 WiFi。
